### 3. 禁止浏览器保存密码

https://www.zhihu.com/question/23529765

当 input 框的 type 为 password 时，浏览器会启用自动保存密码的功能。

将 input 框 type 设置 text,并将字体改为 password,达到类似于密码输入框的效果，且不会被浏览器记录。
